# Student Attendance Management System (ElSewedy School)
 Live website : https://attendance.sewedy.com.eg

 This is a student attendance management website with role based functions and permissions that allows to get analysis from student absences and allows teachers to take student absence during session and exports reports to student affair which saves time when dealing with student absences problems and students can track thier absences through thier accounts
